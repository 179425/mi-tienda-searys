Subir Imágenes a Cloudinary

Las imágenes se almacenan en Cloudinary para no sobrecargar la base de datos

Configuración de Cloudinary:

  1. Crea una cuenta gratis en cloudinary.com
  2. Ve a Settings → Upload → Upload Presets
  3. Crea un nuevo preset con modo "Unsigned"
  4. Copia tu Cloud Name y Upload Preset
  5. Edita el archivo tienda-config.js con tus datos
